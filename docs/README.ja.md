# FoneClaw

FoneClaw は、対応する Android の操作と日常的なスマートフォン作業を音声で支援する独立した Android AI phone assistant です。

Website: https://www.foneclaw.ai/ja/

## Main capabilities

- Phone status and health checks
- Notification and SMS summaries
- Calls and messaging workflows
- System controls such as brightness, volume, Wi-Fi, and Bluetooth
- Screenshot and screen understanding
- Calendar, alarm, notes, maps, and navigation workflows
- Supported app actions by voice

FoneClaw is independent and is not owned by Xiaomi, Google, Samsung, or Apple.
