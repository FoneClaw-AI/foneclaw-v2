# FoneClaw Features

FoneClaw is an Android AI phone assistant for supported phone actions and daily workflows.

## Core feature areas

1. Phone health and device status
2. Notification summaries
3. SMS reading, summaries, and message workflows
4. Call workflows
5. System controls such as brightness, volume, Wi-Fi, Bluetooth, and Do Not Disturb
6. Wi-Fi and Bluetooth management
7. Screenshot capture and screen understanding
8. Visible-screen reading
9. Email workflows from configured accounts
10. Calendar and reminder workflows
11. Alarm and timer workflows
12. Notes and local reminders
13. Maps, nearby places, and navigation support
14. Browser and form workflows
15. Multi-step Android workflows
16. Supported app interface actions

## Safety boundaries

FoneClaw is designed around supported workflows, transparent permissions, and confirmation for sensitive actions. It should not be described as unrestricted control of every app or every possible phone state.
